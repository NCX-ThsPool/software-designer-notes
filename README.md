# 📚 软件设计师备考笔记（中级）

> 本仓库是我备考 **全国计算机技术与软件专业技术资格（水平）考试（软考）中级-软件设计师** 的学习笔记与知识总结。
>
> ⭐ 如果对你有帮助，欢迎点个 Star 支持一下，也欢迎 Fork 和 Issue 交流！

## 📖 仓库简介

本仓库基于官方教程、历年真题及个人学习心得整理而成。内容覆盖考试大纲的核心知识点，包含详细的章节笔记、高频考点总结以及个人刷题过程中的解析思路。

希望能为正在备考的同学们提供一份清晰、可直接查阅的复习资料。

## 🚀 如何使用

- **在线阅读**：直接点击上方目录中的 `.md` 文件，即可在 GitHub 上在线浏览。
- **克隆到本地**：如果你想在本地用 Markdown 编辑器（如 Typora、Obsidian）阅读，可以执行以下命令：

```bash
git clone https://NCX-ThsPool/software-designer-notes.git
```

## 📝 笔记特点

- **考点精炼**：每个章节按“知识点 + 题眼 + 记忆技巧”整理，适合考前快速回顾。
- **真题融合**：笔记中穿插了历年真题的解析，重点说明“题眼是什么，为什么选这个答案”。
- **图表辅助**：大量使用 Mermaid 流程图、思维导图辅助理解复杂概念。
